Proyecto Django — subida a GitHub

Instrucciones rápidas:

1) Ya inicialicé git localmente y añadí un `.gitignore`.
2) Para crear un repositorio remoto en GitHub puedes usar la interfaz web o la CLI `gh`.

Comandos sugeridos (reemplaza `USERNAME` y `REPO`):

```
# crear repo remoto (opcional, con gh cli)
gh repo create USERNAME/REPO --public --source=. --remote=origin

# o añadir remote manualmente
git remote add origin https://github.com/USERNAME/REPO.git

git push -u origin main
```

Si necesitas, puedo (a) crear el repo remoto usando tu token/`gh` CLI, o (b) ejecutar el `git push` si me das la URL remota y credenciales.
